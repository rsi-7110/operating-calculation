# operating-calculation
稼働率計算


このプロジェクトは稼働率の計算を容易にするためのwebツールです。
このプロジェクトのe2eテストをPlaywright MCPを用いて構築してください。
計算結果の整合性を担保するために開始、終了の値でマトリクスを構築してください。

**テスト観点**
- コンポーネントの存在確認
- 入力コンポーネントの内容
- ボタン押下時のイベント発火
- 計算結果の整合性

**画面描画時**
- 「開始」セレクトボックスに`9:00`が入力されていること
- 「開始」セレクトボックスが活性状態であること
- 「終了」セレクトボックスに`18:00`が入力されていること
- 「終了」セレクトボックスが活性状態であること
- 「計算する」ボタンが表示されていること
- 「計算する」ボタンが活性状態であること
- 「終了 現在時刻」ボタンが表示されていること
- 「終了 現在時刻」ボタンが活性状態であること

**「計算する」ボタン押下時**
- 「開始」セレクトボックスの時間と「終了」セレクトボックスの時間の計算結果が稼働の欄に表示されていること
- 休憩控除が更新されてること
- 稼働(丸め前)に純粋な差分が分単位で表示されていること

**「終了 現在時刻」ボタン押下時**
※現在時刻はPlaywrightのグローバルステートを利用して固定設定してください。
- 「開始」セレクトボックスの時間と現在時刻の計算結果が稼働の欄に表示されていること
- 休憩控除が更新されてること
- 稼働(丸め前)に純粋な差分が分単位で表示されていること

**稼働時間計算の仕様**
- 稼働時間は15分刻みで計算されます。
- 5分未満は0分として扱われます。
- 5分以上は15分として扱われます。
- 終了時間に開始時間以前の時間が入力された場合、終了時間は翌日の時間として扱われます。

**休憩控除**
- 当社は以下の時間帯で休憩があり、その時間帯は稼働時間に含まれません。
    - 12:00~13:00
    - 18:00~18:30
    - 21:30~22:00
    - 01:00~01:30
    - 04:30~05:00
